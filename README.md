# Web Scraping

## Description:
#### Web scraping a website (Books to Scrape) designed for web scraping. Collects product information from all products available and organizes them based on category, stored in a CSV file.

#### What it does:
#### - *Scrapes every book from each category*
#### - *Collets the information per book*
#### - *Creates a CSV file per category*
#### - *Downloads the book cover images*

## Features:

#### - Category scraping
#### - Pagination support
#### - Product information extraction
#### - Image downloading
#### - CSV file generation
#### - OOP using Python classes

## Required Setup:

#### - Python
#### - requests package
#### - bs4 package
#### - csv module

## To Run:

#### - Clone repositories
#### - install the required packages as seen under "Required Setup" or in requirements.txt
####   OR
```bash
pip install -r requirements.txt
```
#### - Run the scraper:
#####    - for phase1, use the url: "https://books.toscrape.com/catalogue/a-light-in-the-attic_1000/index.html"
#####    - for phase2, use the url: "https://books.toscrape.com/catalogue/category/books/fantasy_19/index.html"
#####    - for phase3 and phase4, use the url: "https://books.toscrape.com/index.html"

## Skills Gained: 
#### - Web scraping with BeautifulSoup
#### - Sending HTTP requests with Requests
#### - Working with CSV files
#### - Downloading images
#### - Python classes
#### - Git and GitHub
